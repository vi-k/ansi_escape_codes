clearScreenClose

clearScreenClose top-level constant

String const clearScreenClose

Closing tag for Erase in Display (Clear Screen).

Template: {clearScreenOpen}[n]{clearScreenClose}.

Clears part of the screen:

  • If n is 0 (or missing), clear from cursor to end of screen. See clearScreenToEnd.
  • If n is 1, clear from cursor to beginning of the screen. See clearScreenToBegin.
  • If n is 2, clear entire screen and moves cursor to upper left. See clearScreen.
  • If n is 3, clear entire screen and delete all lines saved in the scrollback buffer. See clearScreenWithBuf.

Implementation
